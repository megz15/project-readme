The Readme Project (or Project Readme ) is a user friendly, intuitive and visually pleasing web application built by Meghraj Goswami that serves books and other resources from the notable shadow library LibGen (and by extension Z-Library and Sci-Hub) by utilizing a modified/self-written unofficial API to the service, which by itself provides a minimal and not fully featured API for integration, most if not all of it is documented in Russian.


Responsive design, extremely simple and pleasant to use!


One of the largest online libraries in the world!


Creation of User Profiles allow users to save books in their own Library!


Availability of Community tab to share your experience with the world!

Objective

The primary aim of this project is to create an easily accessible and easily comprehensible web application that allows users to access information in the form of published work, research papers and unattainable documents, and also fiction and non-fiction works that people read for recreation or leisure, all packed in a beautifully crafted modern user interface.

This project helps people who may be financially incapable of buying printed resources and those who reside in areas where particular books are not circulated.

The users don’t need to have top-of-the-line workstations, simple low-end devices that can connect to the internet once in a while and that have support for current web browsers and standards can open the website and use the service. Resources can be downloaded for offline use. The website is fully responsive with support for varying screen sizes, including mobile and laptop screen sizes.
A cross platoform application is also being developed using Flutter for Android, Linux and Windows, with features like local storage and caching.
